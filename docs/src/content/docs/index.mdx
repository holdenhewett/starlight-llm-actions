This is the playground site for `starlight-llm-actions`. Use it to validate
the plugin during development.

## What you should see

Once Phase B is implemented, the dropdown in the page header should let you:

- **Copy this page as Markdown** — to clipboard
- **View it as Markdown** — opens the source in a new tab
- **Save as PDF** — triggers the print dialog
- **Open with `<provider>`** — opens the page in an LLM (ChatGPT, Claude,
  Gemini, GitHub Copilot, Perplexity, T3 Chat, Cursor)
